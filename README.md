# ✍️ Handwritten Digit Recognition

This project recognizes handwritten digits using Computer Vision and Machine Learning.

## Technologies Used
- Python 3.14
- OpenCV
- Scikit-learn
- Streamlit

## Model
- Support Vector Machine (SVM)
- Accuracy: 98.61%

## Features
- Draw digit using mouse
- Real-time prediction
- Confusion matrix analysis

## Run Instructions
```bash
pip install -r requirements.txt
python train.py
streamlit run app.py
